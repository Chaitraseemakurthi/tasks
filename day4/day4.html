HTML
  • HTML stands for Hyper Text Markup Language
  • It was introduced in 1991 by tim berners lee
  • HTML5 was introduced in 2014
  • In HTML, hypertext refers to text containing hyperlinks enabling users to navigate from one page to another page also called routing
  • In HTML, a markup language is used to structure and present content on the web by using tags and attributes to define elements such as headings, paragraphs, links, images, and other multimedia.

Heading tags

Heading tags in HTML, ranging from h1 tag to h6 tag, define headings of different levels, with h1 tag being the highest (most important) level and h6 tag the lowest. They are used to structure content hierarchically

paragraph tag

The paragraph tag in HTML is used to define a paragraph of text. It is a block-level element that creates a distinct section of text with default margins before and after it, facilitating clear organization and readability within a webpage.

Formatting tags
  1. Bold tag : Makes the text bold without adding any extra importance
  2. Italic tag : Makes the text italic without adding any extra emphasis
  3. Strong tag : Makes the text bold and indicates that the text is of strong importance
  4. Emphasize tag : Makes the text italic and indicates that the text is emphasized
  5. Underline tag : Underlines the text
  6. Subscript tag : Renders text as subscript (below the baseline)
  7. Superscript tag : Renders text as superscript (above the baseline)
  8. Deleted tag : Represents deleted text, commonly used to show text that has been removed
  9. Mark tag : It is used to highlight text. The highlighted text typically appears with a yellow background by default.
  10. Blockquote tag : It is used to represent text that is quoted from another website
  11. Abbreviation tag : It is used to define an abbreviation providing a full description when hovered over by the user.
  12. Quataiton tag : It is used to indicate quotations.
  13. Cite tag : It is used to denote the title of a work, such as a book, article, movie, or other creative work. The content within the cite tag is typically rendered in italicized text by default.
  14. Address tag : It is used to represent the address information
  15. Code tag : Represents a fragment of computer code with a monospace font
  16. Keyboard tag : It is used to indicate text that represents keyboard input
  17. Preformatted tag : It preserves whitespace and line breaks for displaying text exactly as it appears in the source code.
Blockline elements

Block-level elements in HTML are elements that typically start on a new line and occupy the full width available to them such as paragraph tag, heading tags and lists

Inline elements

Inline elements in HTML are elements that do not start on a new line and only occupy as much width as necessary, allowing other elements to sit beside them horizontally. Examples include formatting tags and anchor tag.

Table tag
  • It is used to represent content in a tabular form
  • Tables are defined with table tag
  • Rows are represented with tr
  • Columns are represented with td
  • Column headings are represented with th

Table features
  1. Table caption : It is used to provide title for a table
  2. Cell padding : It is used to create spaace around content in a cell
  3. Cell spacing : It is used to create a space between cells
  4. Background color : It is used to apply background color to a table
  5. width and height : It is used to define height and width to an element

Structure of url
  1. Protocol: It is a set of rules and standards that define how data is transmitted and received over a network(e.g., http, https)
  2. Subdomain: A subdivision of the main domain (e.g., www in www.example.com)
  3. Second-Level Domain (SLD): The main domain name (e.g., example in example.com)
  4. Top-Level Domain (TLD): The highest level of the domain name system hierarchy (e.g., com in example.com)
  5. Port: The port number on the server (default ports are 80 for HTTP and 443 for HTTPS).
Image tag

The image tag in HTML is used to embed images into a webpage. It is an empty (self-closing) tag that requires the src attribute to specify the image source URL, and typically includes the alt attribute to provide alternative text if the image cannot be displayed
Anchor tag

The anchor tag is used to create hyperlinks, allowing users to navigate to other webpages, files, or locations within the same page. It typically includes the href attribute, which specifies the URL of the linked resource.
List tag

The ul (unordered list), ol (ordered list), dl(description list) and li tags in HTML are used to create lists. The ul tag creates a bulleted list, the ol tag creates a numbered list, and the li tag defines individual list items within both types of lists.
